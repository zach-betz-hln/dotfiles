# dotfiles

Gluing things together with shell scripts.

## One-Time Setup

```sh
mkdir -p ${HOME}/dev
cd ${HOME}/dev
git clone git@github.com:zach-betz-hln/dotfiles.git
cd dotfiles
./setup.sh
```
